## WWW.STEFFANALOG.COM
My photography website.  
Made with: next.js 13, typescript, graphQL and postgreSQL.  
Investigating ORM ... maybe Sequelize, typeORM or Prisma.

## Getting Started

1. install dependencies:
```bash
pnpm install
```

2. run the development server:
```bash
pnpm dev
```
